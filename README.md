Zudu Automation Project :
This is an automation testing framework created for the Zudu application.

 * Tech Stack
- Selenium WebDriver
- TestNG
- Maven (POM.xml for dependencies)
- Java

* Project Structure
- `src/` → Contains test scripts and page objects  
- `pom.xml` → Maven dependencies and build management  
- `Testng.xml` → Test suite configuration  
- `files/` → Test data (if applicable)

*  How to Run
1. Open the project in Eclipse or IntelliJ
2. Update drivers/config if needed
3. Run `Testng.xml`

This is repository showcases end to end aotomation testing for zudu app including login , agent page flow and batch call testing flow.
